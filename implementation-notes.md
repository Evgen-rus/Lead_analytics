# Implementation notes: экспорт аналитики в Google Таблицы

## Утверждённые решения

- Добавить создание Google Таблицы непосредственно в приложение Lead Analytics.
- Использовать service-account credential по пути из `GOOGLE_CREDENTIALS_FILE`.
- Создавать таблицы в общей папке Drive `1jZZAuQqHFVQfDrfbKjrAPSAyTN4_SjLd`.
- После упрощения сценария всегда записывать в существующую таблицу `1MI68uPNNW5GqkxQ7MLMWHM-WHGuZZmsjrNfvWLNHRYk`, создавая новые вкладки.
- Сначала проверить создание пустой тестовой таблицы без выгрузки клиентских данных.
- Не читать и не выводить содержимое credential-файла.

## Отклонения от плана

- Пустая тестовая таблица не создана: Google Sheets API вернул `403 The caller does not have permission`.
- Пользователь выбрал более простой сценарий: не создавать новые файлы на Drive, а добавлять вкладки в одну существующую таблицу.

## Найденные edge cases

- Доступ service account к существующим таблицам не гарантирует право создавать новые файлы или доступ пользователя к созданному файлу.
- Для создания таблицы внутри общей папки должен быть отдельно включён Google Drive API в Google Cloud project service account.
- Повторная выгрузка того же проекта и даты не перезаписывает вкладки, а получает суффикс `(2)`, `(3)` и далее.
- Имя вкладки очищается от запрещённых символов и ограничивается 100 символами.

## Консервативные допущения

- Имя тестовой таблицы содержит название приложения, пометку теста и текущую дату.
- На первом тесте создаётся пустая таблица без аналитических данных.
- Экспорт запускается отдельной кнопкой после готовности анализа, а не автоматически.
- Заголовок оформляется единообразно, первая строка закрепляется, таблица получает фильтр и автоширину колонок.
- Для будущих выгрузок фактические цветовые заливки ячеек переносятся из готового Excel; старые Google-вкладки не изменяются.
- Создание и оформление всех вкладок объединено в batchUpdate, значения группируются в values.batchUpdate до 1.8 МБ.
- Внутри приложения Google-экспорты выполняются последовательно; для `429/5xx` действует до пяти попыток с экспоненциальной задержкой.

## Закрытые неизвестные

- В проекте есть только заглушка интеграции Google Sheets.
- Credential-файл существует по указанному относительному пути.
- Авторизация до Google API проходит, но service account не может самостоятельно создать таблицу через `spreadsheets.create`.
- Service account успешно создал вкладку `Тест подключения 15.07.2026` в целевой таблице и записал тестовые значения.

## Оставшиеся неизвестные

- Нужен ли в будущем отдельный фоновой job для очень больших Google-экспортов.

## Блокеры

- Нет для записи в существующую Google Таблицу; Google Drive API в выбранном сценарии не используется.
